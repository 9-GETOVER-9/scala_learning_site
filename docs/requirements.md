# Scala 课程交互式学习网站需求摘要

## 项目目标

建设一个以章节为一级目录、以每节课为学习单元、以知识点为核心模块的交互式学习网站。

## 页面结构

- 首页：负责课程搜索、筛选和入口展示。
- 课程详情页：负责展示课堂摘要、知识点、代码示例和互动练习。
- Java 互动练习专区：保留已有 Java 小游戏入口。

## 首页功能

- 按章节筛选
- 按上课日期筛选
- 按知识点筛选
- 关键词搜索
- 显示课程卡片
- 显示 Java 互动练习入口

## 课程详情页模块

- 课程基本信息
- 课堂摘要
- 知识点拆解
- 代码示例
- 小测验
- 拖拽匹配
- 流程动画
- 练习题
- 课后总结

## 技术方案

采用纯 HTML / CSS / JavaScript。

原因：

- 适合 GitHub Pages 部署
- 不需要后端
- 不需要数据库
- 后续更新简单
- 小白更容易维护

## 后续更新方式

以后新增课程时，主要修改 `js/data.js` 文件，把新课整理成一条课程数据即可。
