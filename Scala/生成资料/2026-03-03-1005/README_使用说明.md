# 使用说明

本文件夹包含根据 `3 3.md` 整理出的两份内容：

1. `scala-2026-03-03-1005-summary.md`
   - 摘要后的 Markdown 版本
   - 可作为课程笔记、资料备份、提交说明使用

2. `scala-2026-03-03-1005-data-object.js`
   - 可直接追加到之前网站项目的 `js/data.js` 里的课程对象
   - 添加后访问地址为：
     `lesson.html?id=scala-2026-03-03-course-intro`

添加方式：

```js
const lessons = [
  {
    // 原来的课程
  },
  {
    // 把 data-object.js 中的整个对象粘贴到这里
  }
];
```

注意：
- 如果前面已有课程对象，前一个对象后面要加英文逗号。
- 这是 2026-03-03 10:05 的课程，和 11:00 的 Scala 基础课应作为两节不同课程。
