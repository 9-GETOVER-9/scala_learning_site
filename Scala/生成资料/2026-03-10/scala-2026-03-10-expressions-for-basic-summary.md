## 一、课堂摘要

本节课主要讲解 Scala 中表达式、block、for 循环、while 循环和函数定义等内容，并结合职业发展、AI 时代岗位变化和学科交叉进行扩展说明。

老师首先强调 Scala 中表达式与 Java 语句的区别。表达式有值，可以被变量或常量接收；Scala 中 block 的最后一行通常就是返回值，这与 Java 中依赖 `return` 的传统方法不同。

随后课堂讲解了两点间距离的计算示例。在 Java 中通常需要定义方法、传递参数；而 Scala 可以把计算步骤放到 block 中，通过最后一行直接得到计算结果，体现了模块化和表达式化的编程思想。

课堂后半部分重点讲解 Scala 的 for 循环。Scala 可以在括号中定义变量，通过左箭头 `<-` 在 range 范围中遍历。与 Java 相比，Scala 的 for 循环写法更简洁，尤其在连续、同类型数据遍历时，不需要像 Java 一样先建立数组。

此外，老师还结合 Open Cloud、智能体、数据架构师、算法工程师等话题，提醒同学未来岗位更看重数学能力、编程思想、大数据算法和垂直应用能力。

## 二、核心知识点

### 1. 表达式与语句的区别

Java 中很多内容被称为语句或方法，而 Scala 更强调表达式。

表达式的特点：

- 有值；
- 可以赋值；
- 可以被变量或常量接收；
- 可以作为计算结果继续使用。

示例：

```scala
val result = if (x > 0) 1 else -1
```

这里整个 `if...else` 都是表达式，最终结果可以赋给 `result`。

---

### 2. block 的最后一行是返回值

Scala 中 block 通常用大括号表示。

```scala
val value = {
  val a = 10
  val b = 20
  a + b
}
```

解释：

- 前两行是计算过程；
- 最后一行 `a + b` 是 block 的值；
- `value` 最终等于 `30`。

---

### 3. Java 与 Scala 求距离的思想区别

Java 中求两点间距离通常需要：

- 定义方法；
- 定义参数；
- 传递参数；
- 使用 return 返回结果。

Scala 中可以直接用 block 表达计算过程：

```scala
val distance = {
  val dx = x - x0
  val dy = y - y0
  math.sqrt(dx * dx + dy * dy)
}
```

这种方式更像“把计算模块放进一个块里”，最后直接得到值。

---

### 4. for 循环基本形式

Scala 的 for 循环基本结构：

```scala
for (i <- 1 to 5) {
  println(i)
}
```

解释：

- `i` 是循环变量；
- `<-` 表示从右边范围中依次取值；
- `1 to 5` 表示从 1 到 5；
- 循环体中执行 `println(i)`。

---

### 5. range 范围

Scala 中 `range` 可以理解为一个范围或区间。

常见写法：

```scala
1 to 5
1 until 5
```

区别：

| 写法 | 含义 |
|---|---|
| `1 to 5` | 包含 5 |
| `1 until 5` | 不包含 5 |

---

### 6. while 循环

Scala 的 while 循环和 Java 类似。

```scala
var i = 1

while (i <= 5) {
  println(i)
  i = i + 1
}
```

当条件为 true 时，执行循环体。

---

### 7. def 是定义函数的关键字

Scala 中使用 `def` 定义函数。

```scala
def signum(a: Int): Int = {
  if (a > 0) 1 else if (a == 0) 0 else -1
}
```

解释：

- `a: Int` 表示参数 `a` 是整数；
- 函数名后面的 `: Int` 表示返回值是整数；
- 函数体最后一行决定返回结果。

---

### 8. Scala 中类型放在变量后面

Scala 中参数或变量类型通常放在名字后面：

```scala
a: Int
args: Array[String]
```

而 Java 中类型通常放在名字前面。

---

### 9. Scala for 循环比 Java 更简洁

Java 中遍历数据通常需要：

- 建立数组；
- 写循环条件；
- 控制下标；
- 处理数组空间。

Scala 中如果数据是连续且类型相同的范围，可以直接写：

```scala
for (i <- 1 to 100) {
  println(i)
}
```

这体现了 Scala 的简洁性。

---

### 10. 二维遍历

Java 中二维数组常用双重循环。

Scala 中可以在一个 for 结构里写多个生成器：

```scala
for (i <- 1 to 3; j <- 1 to 3) {
  println(i + "," + j)
}
```

也可以用逗号或分号分隔多个生成器，表达多重循环。

## 三、代码示例

### 示例 1：if 表达式赋值

```scala
val result = if (x > 0) 1 else -1
```

解释：`if...else` 整体有值，可以赋给变量。

---

### 示例 2：block 计算距离

```scala
val distance = {
  val dx = x - x0
  val dy = y - y0
  math.sqrt(dx * dx + dy * dy)
}
```

解释：最后一行 `math.sqrt(...)` 是整个 block 的返回值。

---

### 示例 3：for 循环遍历范围

```scala
for (i <- 1 to 5) {
  println(i)
}
```

解释：输出 1 到 5。

---

### 示例 4：while 循环

```scala
var i = 1

while (i <= 5) {
  println(i)
  i = i + 1
}
```

解释：只要条件成立，就持续执行循环体。

---

### 示例 5：函数定义

```scala
def signum(a: Int): Int = {
  if (a > 0) 1 else if (a == 0) 0 else -1
}
```

解释：这个函数根据参数正负返回 1、0 或 -1。

---

### 示例 6：二维遍历

```scala
for (i <- 1 to 3; j <- 1 to 3) {
  println(i + "," + j)
}
```

解释：相当于 Java 中的双重循环。

## 四、小测验

### 题目 1

Scala 中表达式的最大特点是什么？

A. 没有值  
B. 有值，可以赋值  
C. 只能写注释  
D. 只能输出字符串  

**正确答案**：B  
**解析**：Scala 中表达式有值，可以被变量或常量接收。

---

### 题目 2

Scala 中 block 的返回值通常是哪一行？

A. 第一行  
B. 最后一行  
C. 注释行  
D. 空行  

**正确答案**：B  
**解析**：Scala 中 block 的最后一行通常作为整个 block 的值。

---

### 题目 3

Scala 中 for 循环的 `<-` 可以理解为什么？

A. 从右边范围取值给左边变量  
B. 删除变量  
C. 创建数据库  
D. 表示字符串拼接  

**正确答案**：A  
**解析**：例如 `i <- 1 to 5` 表示 i 依次从范围中取值。

---

### 题目 4

`1 to 5` 是否包含 5？

A. 包含  
B. 不包含  
C. 只包含 0  
D. 只包含 6  

**正确答案**：A  
**解析**：`to` 表示闭区间，包含右端点。

---

### 题目 5

Scala 中定义函数使用哪个关键字？

A. var  
B. val  
C. def  
D. object  

**正确答案**：C  
**解析**：Scala 使用 `def` 定义函数。

## 六、流程动画建议

### 流程 1：表达式如何赋值

```text
写出一个表达式
↓
表达式进行计算
↓
得到一个值
↓
用 val 或 var 接收这个值
↓
后续可以继续使用这个变量
```

### 流程 2：for 循环执行过程

```text
看到 for 循环
↓
读取右侧 range 范围
↓
将第一个值赋给循环变量
↓
执行循环体
↓
继续取下一个值
↓
直到范围遍历完成
```
