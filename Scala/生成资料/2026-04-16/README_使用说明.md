# 使用说明

本压缩包根据 `4 02.zip` 中的 `4 16.md` 整理生成。

## 文件说明

1. `scala-2026-04-16-pattern-case-classes-summary.md`
   - 4 月 16 日课程的摘要版 Markdown

2. `scala-2026-04-16-pattern-case-classes-data-object.js`
   - 4 月 16 日课程的数据对象

3. `README_使用说明.md`
   - 本说明文件

## 添加位置

把 `scala-2026-04-16-pattern-case-classes-data-object.js` 文件里的整个对象复制到：

```text
scala_learning_site/
└── js/
    └── data.js
```

添加到 `const lessons = [...]` 数组中。

## 访问地址

```text
lesson.html?id=scala-2026-04-16-pattern-case-classes
```

## 注意

如果 `lessons` 数组里前面已有课程对象，前一个对象后面要加英文逗号。
