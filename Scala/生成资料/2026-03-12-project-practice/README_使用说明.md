# 使用说明

本压缩包根据你上传的 `3 12.md` 整理生成。

## 文件说明

1. `scala-2026-03-12-project-practice-summary.md`
   - 摘要后的 Markdown 版本
   - 可作为课程笔记、资料备份或提交说明

2. `scala-2026-03-12-project-practice-data-object.js`
   - 可直接追加到之前网站项目的 `js/data.js` 的 `lessons` 数组中

## 添加位置

```text
scala_learning_site/
└── js/
    └── data.js
```

把 `scala-2026-03-12-project-practice-data-object.js` 里的整个对象复制进去。

## 访问地址

添加后，本节课页面地址为：

```text
lesson.html?id=scala-2026-03-12-project-practice
```

## 注意

如果 `lessons` 数组里前面已有课程对象，前一个对象后面要加英文逗号。
