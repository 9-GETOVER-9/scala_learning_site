【智能总结】
本次课堂主要讲解了Scala与Java在类、对象、方法、字段及构造器等方面的编程知识差异。
- **类和对象基础**：
    - **定义区别**：Java 中执行入口在测试类，Scala 中对象常作应用程序入口，且 Scala 更灵活。
    - **无参方法**：无参方法后无括号，使用时需初始化字段，方法默认公有。
- **运算符差异**：Scala 无自加自减运算符，需自定义方法实现加一或减一功能。
- **方法调用规则**：
    - **参数对应**：方法调用时，若方法有参数，调用处需对应传参；无参数则省略。
    - **改置器和取值器**：改置器（如 set 方法）改变对象状态，带括号传参；取值器（如 get 方法）不改变，不带括号。
- **Scala 字段特性**：
    - **公有字段**：Scala 对每个字段自动提供 get 和 set 方法，字段缺修饰符时为公有。
    - **私有字段**：私有字段的 get 和 set 方法默认私有，可重写；常量字段只有取值器。
- **主构造器与辅助构造器**：
    - **主构造器**：Scala 中类本身即主构造器，参数放类名后，编译阶段自动提供 get 和 set 方法。
    - **辅助构造器**：辅助主构造器构造对象，第一行基于主构造器，类似继承中调用父类构造方法。
