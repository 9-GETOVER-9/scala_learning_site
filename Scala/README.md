# Scala 课程资料说明

这个目录集中保存 Scala 课程的原始纪要、整理后总结和用于网站课程数据的生成资料。

## 目录作用

### `课堂纪要/`

保存根据课堂录音或课堂内容整理的详细纪要。这些文件信息较完整，主要用于回顾原课程和核对知识点。

命名格式：

```text
课堂纪要 M-D 课程主题-上.md
课堂纪要 M-D 课程主题-下.md
```

如果当天只有一份内容，可以不使用“上/下”后缀。

### `课程总结/`

保存从课堂纪要中提炼出的结构化课程总结，适合快速复习、查找示例和生成网站课程内容。

命名格式：

```text
课堂总结 M-D Scala课程主题-上.md
课堂总结 M-D Scala课程主题-下.md
```

### `生成资料/`

保存按课程日期分批生成的中间产物。每个批次通常包含：

- `*-summary.md`：该批次的 Markdown 课程摘要。
- `*-data-object.js`：可合并到网站主数据的课程对象。
- `README_使用说明.md`：该批次的导入说明。

这些 JavaScript 文件是生成资料，不会被网页直接加载。网站实际读取的课程主数据位于项目根目录的 `js/data.js`。

## 新增课程建议流程

1. 将详细内容保存到 `课堂纪要/`。
2. 将提炼后的复习笔记保存到 `课程总结/`。
3. 如果生成了网站数据对象，将完整批次保存到 `生成资料/YYYY-MM-DD-主题/`。
4. 检查课程 `id` 不重复后，再将数据对象合并到 `../js/data.js`。
5. 打开首页和课程详情页，检查搜索、筛选和课程内容。
