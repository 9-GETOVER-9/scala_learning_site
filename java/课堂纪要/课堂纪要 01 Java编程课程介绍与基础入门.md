本次课程介绍Java编程课程要求、安排及项目创建，讲解Java特点、地位与基础语法。

\- \*\*课程要求\*\*：

\- \*\*学习态度\*\*：课程重要，需认真对待，上课不溜号、不玩手机。

\- \*\*创建项目\*\*：在Java SE中创建项目，项目名无空格，可加下划线，运行环境选Live版本，module information处点击Don't create。

\- \*\*创建类\*\*：在SRC文件夹右键选New - class，类中属性和行为构成实体，创建类需默写特定语句作为入口。

\- \*\*Java特点与地位\*\*：开发企业应用APP，简单方便，跨平台可移植，课程使用Java SE平台。

\- \*\*Java基础语法\*\*：

\- \*\*修饰词\*\*：类的修饰词有public、private、protected、default四种，初期都写成public。

\- \*\*主函数\*\*：public static void main(String args\[\])是主函数，为入口函数，由修饰符、静态、返回值类型、函数名、参数列表组成，需牢记。

\- \*\*函数组成\*\*：由方法头和方法体组成，system out println是输出语句。
