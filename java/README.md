# Java 课程资料说明

这个目录集中保存 Java 课程的详细纪要、结构化总结和网站课程数据生成批次。

## 目录作用

### `课堂纪要/`

保存 19 节 Java 课程的详细课堂纪要。文件编号和标题与 `课程总结/` 一一对应。

命名格式：

```text
课堂纪要 NN Java课程主题.md
```

`NN` 使用两位数，例如 `01`、`02`、`19`。

### `课程总结/`

保存从同编号课堂纪要中提炼出的复习资料。每份总结与纪要使用相同的课程编号和标题。

命名格式：

```text
课堂总结 NN Java课程主题.md
```

### `课程数据/`

保存分批生成的 JavaScript 课程对象。内部的 `java_course_data_objects*` 目录代表不同导入批次，保留批次名是为了便于追溯和核对。

这些 JavaScript 文件不会被网页直接加载。网站实际使用的 Java 和 Scala 课程数据统一保存在项目根目录的 `js/data.js`。

## 新增课程建议流程

1. 为新课程选择下一个两位编号。
2. 在 `课堂纪要/` 和 `课程总结/` 中使用相同编号与标题。
3. 将生成的网站数据对象保存到 `课程数据/` 中的新批次目录。
4. 检查课程 `id` 不重复后，再将对象合并到 `../js/data.js`。
5. 打开首页和对应课程页，确认课程卡片、搜索和详情内容正常。
